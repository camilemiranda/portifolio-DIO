Este é o meu Portfólio Inicial!

Bem-vindo(a) ao meu portfólio de testes! Este espaço foi criado para organizar e exibir meus primeiros projetos de desenvolvimento web e planilhas interativas. Aqui você encontrará exemplos de HTML, CSS, JavaScript e Excel que usei para praticar, aprender e experimentar ideias. Meu objetivo é mostrar meu processo de evolução e como transformo conceitos em projetos funcionais.



Cada projeto apresentado neste portfólio serve como uma oportunidade de explorar novas técnicas, testar layouts, praticar boas práticas de codificação e experimentar diferentes abordagens de design. Sinta-se à vontade para explorar, clicar nos projetos, conhecer minhas habilidades e acompanhar a minha evolução. Este portfólio é meu laboratório criativo, onde aprendo, erro, ajusto e cresço em cada projeto.

Meus Projetos

LEADING PAGE | DIO

Meu primeiro projeto prático em HTML e CSS, desenvolvido como parte do desafio da DIO. O objetivo foi construir uma landing page do zero, aplicando fundamentos essenciais de estilização, como uso de propriedades básicas do CSS, unidades de medida relativas e absolutas, e organização semântica do HTML. Esse projeto marcou o início da minha jornada no Front-End, reforçando boas práticas e a base para evoluir em layouts mais complexos e responsivos

CLOLE DISCORD

Projeto desenvolvido para recriar a landing page do Discord utilizando HTML5 e CSS3. O desafio teve como foco a estrutura semântica, a responsividade com media queries e o uso de Flexbox e Grid para organização visual. O resultado é uma página fiel ao layout original, com botões interativos e adaptação para dispositivos móveis de até 428px. Esse projeto consolidou minha prática em layouts responsivos e reforçou conceitos essenciais de organização de código e estilização moderna.

CLONE YouTube

Projeto desenvolvido para recriar a interface do YouTube utilizando HTML5 e CSS3. O desafio teve como foco a estrutura semântica, a responsividade com Media Queries e o uso de Flexbox e Grid para organização visual. O resultado é uma página fiel ao layout original, com cabeçalho fixo, sidebar de navegação, player de vídeo e lista de vídeos sugeridos, adaptando-se a desktops, tablets e dispositivos móveis.

IR SYNC

Projeto desenvolvido para criar uma planilha de controle de dados do Imposto de Renda em Microsoft Excel. O foco esteve na organização das informações, clareza visual e praticidade, com abas para dados do titular, rendimentos e notas bancárias. O resultado é uma planilha estilizada, intuitiva e funcional, que permite centralizar e controlar dados essenciais da declaração de forma eficiente. Esse projeto consolidou minha prática em estruturação de planilhas, organização de dados e design visual funcional.

ARK INVEST

Projeto desenvolvido para criar um simulador de investimentos em Fundos Imobiliários (FIIs) utilizando Microsoft Excel. O foco esteve na criação de uma ferramenta funcional, visual e interativa, com cálculo automático de aportes, dividendos, rendimento e patrimônio acumulado ao longo do tempo. O resultado é uma planilha intuitiva que permite simular cenários de investimento, acompanhar o crescimento do patrimônio e planejar estratégias de acordo com diferentes perfis de investidor..
